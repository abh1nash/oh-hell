# Oh Hell! Card Game

Very basic implementation of the card game _Oh Hell!_ for upto seven people.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Run server

```
npm run server
```

### Compiles and minifies for production

```
npm run build
```

### For deployment

```
npm run start
```
